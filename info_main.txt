SETTINGS
Quick - same timer for all images
Class - auto-distribute by tiers (manual distribution available in file manager)
Set session time limit - lmk forward, rmk back (no limit by default)

Pin - always on top (slideshow only)
Palette - accent color
Sun/Moon - toggle dark and light theme

Image button - open file manager. Drag and drop to upload files or use file, folder or url buttons respectively. 
Google drive and yandex disk support available (folders/files must be public). Files remain in cache until manual clearing on the bin button (appears when filled). 
Select multiple files through shift or ctrl. Delete to..delete. 
Eraser button clears the file list. Toggle shuffle to unlock manual reorder. 
Rmk to move files to a different or pin in a  current time group. 
File manager can be dettached and resized. Drag to the main window on any side to reattach.
