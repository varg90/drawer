What's new in v7

Headers aligned

Main and editor now share the same top padding (12px) and side padding (14px). When snapped, the header icons sit at the exact same height and horizontal margin across both windows. No more visual mismatch.

Rebalanced top

With accent bar gone, the top padding increased from 10px to 12px. Gives the header breathing room from the window edge without a decorative element. The rounded corners + padding create enough visual framing on their own.

Dresser panel

2px warm border frames the recessed panel like a furniture compartment. The panel stretches to fill available space — modes and timer buttons center vertically inside. No more dead space below; the compartment absorbs it. Timer buttons = drawers in a dresser.

Lexend font

Designed for readability and dyslexia support. Wider letterforms, distinct characters (no b/d confusion), generous spacing. Warm and friendly. 300-500 weight for body, 500 for labels. Title stays Lora serif. ~60KB bundled.

Rounded windows

8px radius. When snapped: main rounds left, editor rounds right — shared flat edge. Standalone: all corners round. In PyQt6: WA_TranslucentBackground + QPainter rounded rect mask. Snap logic adjusts corners per dock state.

Rounding scale

3px editor list items, 5px buttons/tabs/add, 6px inset panel, 8px windows, 10px start button. The outlined panel at 6px sits between the button scale and the window scale — a clear visual layer.
